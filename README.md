# Gallery App

Мини-проект на React + Vite с галереей изображений, загрузкой данных через API и маршрутизацией.

## Что есть в проекте

- главная страница с локальной галереей, фильтрацией, поиском и CRUD-формой
- отдельная API-страница с карточками, загруженными через `fetch`
- сохранение локальной коллекции в `localStorage`
- маршруты на базе `react-router-dom`

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Основные файлы

- `src/App.jsx` - общее состояние, `useEffect`, маршруты и обработчики
- `src/pages/GalleryPage.jsx` - основная страница галереи
- `src/pages/ApiPage.jsx` - отдельная страница с данными из API
- `src/components/*` - переиспользуемые UI-компоненты
- `src/data/seedShots.js` - стартовые данные
