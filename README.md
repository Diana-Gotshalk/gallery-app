+ Галерея с фильтрами, каруселью и CRUD

Небольшое приложение на React + Vite: фильтр по категориям, поиск, карусель выбранного кадра, сетка превью, добавление/редактирование/удаление снимков.

## Возможности
- Фильтрация по категории + поиск по названию/описанию/локации.
- Карусель с кнопками «вперёд/назад», счётчиком, метаданными.
- Сетка превью, выбор карточки кликом.
- Формы для добавления и редактирования, удаление карточек.

## Требования
- Node.js 20.19+.

## Установка и запуск
```bash
git clone https://github.com/<user>/<repo>.git
cd <repo>
npm install
npm run dev
```
Открыть адрес из вывода (обычно `http://localhost:5173/`).

## Сборка
```bash
npm run build
```
Готовые файлы — в `dist/`.

## Структура
- `src/App.jsx` — состояние и сборка страницы.
- `src/components/*` — отдельные компоненты (фильтр, поиск, карусель, сетка, форма).
- `src/App.css`, `src/index.css` — стили.
